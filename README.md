## Proyecto: Showcase de CSS moderno

### Objetivo de la práctica

Desarrollar un proyecto web demostrativo cuyo objetivo sea aplicar y mostrar el uso de CSS moderno, trabajando:

- **Pseudoclases**
- **Animaciones CSS**
- **Estilos avanzados con imágenes**
- **Iconos con Font Awesome**
- **Tipografías externas**
- **Diseño responsive**

Esta práctica no se plantea como un trabajo "para el profesor", sino como un **proyecto personal y profesional**, pensado para que el alumno pueda mostrarlo públicamente como parte de su portfolio.

---

## Enfoque profesional del proyecto (MUY IMPORTANTE)

Este proyecto debe entenderse como:

- Una pieza de portfolio personal

## Requisitos técnicos OBLIGATORIOS

### HTML

- HTML5 bien estructurado
- Uso correcto de:
  - `<header>`
  - `<main>`
  - `<section>`
  - `<footer>`
- Código ordenado e indentado

---

### CSS moderno (OBLIGATORIO)

#### Pseudoclases

Uso de **al menos 4 pseudoclases distintas**, por ejemplo:

- `:hover`
- `:focus`
- `:active`
- `:nth-child()`
- `:first-child` / `:last-child`

**Su efecto debe ser visible.**

#### Animaciones CSS

- Al menos **1 animación con @keyframes**
- **Uso de:**
  - `animation`
  - `transition`
- Animaciones suaves y bien integradas

#### Imágenes y efectos visuales

- Uso de imágenes (fondo o contenido)
- **Uso obligatorio de `mix-blend-mode`**
- **Uso de:**
  - `background-image`
  - `background-size`
  - `background-position`

**Opcional (valorado):**

- `filter`
- `opacity`
- `clip-path`

#### Tipografías

- **Uso obligatorio de tipografías externas (Google Fonts)**
- Diferenciar títulos y texto normal

#### Iconos (OBLIGATORIO)

- **Uso exclusivo de Font Awesome**
- Iconos integrados en:
  - Botones
  - Tarjetas
  - Secciones
- **No usar imágenes como iconos**

#### Diseño Responsive (OBLIGATORIO)

- Adaptación correcta a:
  - Escritorio
  - Móvil
- **Uso de:**
  - Media queries
  - Unidades relativas (%, vw, vh, rem)
